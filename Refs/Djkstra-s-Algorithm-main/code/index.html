Dijkstra Solver

Click to create node

Algorithmic Steps:

  1. Set all vertices distances = infinity except for the source vertex, set the source distance = 0.
  2. Insert the source vertex in a array in the form (distance , vertex), as the comparison in the array comparison will be according to vertices distances.
  3. Vertex with the minimum distance from the Source node is searched to insert in the array after the souce node.
  4. Update the distances of the connected vertices to the array in case of "current vertex distance + edge weight < next vertex distance", then insert the vertex with the new distance to the array.
  5. If the index node is visited before, just continue without using it.
  6. Apply the same algorithm again until the array is empty.
